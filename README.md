# Ronaly Garabato — Portfolio

## Deploy to Vercel
1. Go to https://vercel.com and log in (or sign up free).
2. Click "Add New" → "Project" → "Deploy" (or drag-and-drop this whole folder into the Vercel dashboard upload area).
3. No build settings needed — this is a static site (just index.html + assets/).
4. Click Deploy. Vercel will give a live link like ronaly-portfolio.vercel.app.

Alternative (no account needed to test): open index.html directly in any browser.
